# 🏦 Smart Lender - Loan Approval Prediction System

## 📌 Project Overview

Smart Lender is a Machine Learning-based web application that predicts whether a loan application is likely to be approved based on applicant details. The system uses a trained ML model and a Flask web application to provide instant loan approval predictions.

---

## 🚀 Features

- User-friendly web interface
- Predicts loan approval instantly
- Machine Learning model using Random Forest
- Flask-based web application
- Responsive and modern UI
- Easy deployment using Render

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/AP24110010056/Smart-Lender.git
```

2. Open the project folder

```
cd Smart-Lender
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```
